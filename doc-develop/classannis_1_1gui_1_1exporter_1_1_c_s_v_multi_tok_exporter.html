annis.gui.exporter.CSVMultiTokExporter Class Reference

Detailed Description

A csv-exporter that will export the text of the underlying token instead of the base text.

This is useful for getting text spans where the normal csv-exporter doesn't work since there are multiple speakers or normalizations.
