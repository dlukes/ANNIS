annis.sqlgen.IslandsPolicy Class Reference

Detailed Description

Matched nodes that are too far apart to be shown in the same context (distance > context_left+context_right) normally create islands ("..." is displayed between the tokens).

This class contains an enum which is used to configure the behavior of ANNIS in this case.
