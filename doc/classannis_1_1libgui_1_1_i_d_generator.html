annis.libgui.IDGenerator Class Reference

Detailed Description

Helper class to generate unique IDs for components.

Each component will have the ID of it's parent component as prefix in the ID. Thus assigning an ID to an component will always assign an ID to all it's ancestors as well. Additionally it is checked that the siblings of a component don't have the same ID when it is assigned.
